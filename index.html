Triple Triad

Triple Triad est un jeu de cartes que j’ai développé dans le cadre de mon stage de fin d’études, afin de l’intégrer à la plateforme de divertissement Evan destinée aux téléviseurs LG. Le projet a notamment mis l’accent sur la compatibilité avec les versions legacy des navigateurs, garantissant une expérience fluide pour tous les utilisateurs. À noter : le jeu a été conçu spécifiquement pour les téléviseurs LG, principalement utilisés dans les hôpitaux. Il est donc possible que l’expérience sur PC ne fonctionne pas correctement, malgré les solutions techniques habituelles permettant une compatibilité multi-plateformes.
